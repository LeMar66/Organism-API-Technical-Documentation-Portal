esp_err_t Hardware::BusManager::begin ( )

Initialize all communication buses.

This method orchestrates the startup sequence of I2C, SPI, and UART. Failure of a critical bus will return ESP_FAIL.

Returns
esp_err_t ESP_OK on success, ESP_FAIL if a critical bus fails.
